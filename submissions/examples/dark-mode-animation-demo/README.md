# Dark Mode Animation Demo

## What does this do?

This example demonstrates how EaseMotion CSS animation classes can be used inside a modern dark mode interface.

## How is it used?

Open `demo.html` in any browser. The demo showcases:

- Fade-in animation
- Slide-up animation
- Hover grow effect
- Gradient animated heading

## Why is it useful?

Many modern websites support dark mode. This example helps contributors understand how EaseMotion animations look on dark backgrounds while maintaining readability and smooth interactions.
